### local에서 실행 방법

```bash
1. git clone https://github.com/Team-fast4ward/market-log.git
2. cd market-log
3. .env 파일 생성 후, api key(BASE_URL, USER_NAME, API_KEY, ADMIN_EMAIL) 입력
4. npm i && npm run dev
```
